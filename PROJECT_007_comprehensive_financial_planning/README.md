# PROJECT_007_comprehensive_financial_planning

## 综合理财规划 — 张先生案例

**客户画像**：张先生，45岁，家庭年收入 $800K，流动资产 $5M

**项目目标**：生成专业级综合理财规划报告，涵盖资产配置、退休规划、保险需求、税务考量与风险评估。

## 项目结构

```
PROJECT_007_comprehensive_financial_planning/
├── data/              ← 客户财务数据
├── analysis/          ← 核心分析脚本
├── templates/         ← 报告模板
├── output/            ← 生成的报告
├── requirements.txt   ← Python 依赖
└── README.md          ← 本文件
```

## 技术栈

- Python 3.13
- Pandas — 数据处理
- Plotly — 交互式图表
- Jinja2 — HTML 模板渲染

## 输出格式

- 暗色主题 HTML 报告（参考 PROJECT_006 风格）
- 全中文输出，章节使用中文序号
- 专业术语括注英文原文

## 运行方式

```bash
pip install -r requirements.txt
python analysis/financial_analysis.py
```
