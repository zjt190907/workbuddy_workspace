# 加密备忘录桌面精灵 (CryptoMemoPet)

## 简介

桌面常驻加密小精灵，安全保管密保和密码。所有敏感数据 Fernet 加密存储，需要查看时才输入主密码。

## 功能

- **桌宠模式**：启动即显示桌宠，无需密码
- **加密记事本**：右击选择分类 → 输入主密码 → 查看/编辑数据
- **两类数据**：密保问答、账户密码
- **真彩色动画**：PNG 逐帧动画，完整保留颜色（非 GIF 256 色限制）
- **可拖拽**：左键拖拽移动位置
- **可缩放**：鼠标滚轮放大/缩小（50% ~ 300%）
- **敏感字段二次验证**：查看/复制密码需再次输入主密码

## 使用方式

### 启动（无控制台）

双击 `run.bat` 即可启动，桌宠直接出现在桌面上。

右击桌宠：
- 📝 密保问答 — 输入密码后查看/编辑
- 🔑 账户密码 — 输入密码后查看/编辑
- ❌ 退出

滚轮操作：
- 🔼 滚轮上 — 放大精灵
- 🔽 滚轮下 — 缩小精灵

## 数据存储

- 加密数据文件：`%APPDATA%/CryptoMemoPet/vault.dat`
- PBKDF2(600000次) + Fernet(AES-128-CBC) 加密
- 5次密码错误锁定15分钟
- 纯本地应用，无网络通信

## 自定义精灵

### 方式一：帧序列动画（推荐，真彩色）

将 PNG 帧序列放到 `assets/frames/` 目录，命名格式 `f000.png`、`f001.png`、`f002.png` ...
- 每张图自动处理透明背景（白色/浅灰色 → 透明）
- 真彩色显示，无 GIF 256 色限制

### 方式二：单图片

将图片放到 `assets/` 目录，优先级：`pet.gif` > `pet.png` > `pet.jpg` > `pet.webp`

- 静态图会附带呼吸漂浮效果
- 动画 GIF 支持逐帧播放（但颜色可能因调色板受限）

## 技术栈

- Python 3.13.12 + tkinter
- cryptography (加密)
- Pillow (图片处理)
- opencv-python-headless (视频帧提取)
- pywin32 (系统托盘)
