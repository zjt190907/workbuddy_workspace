# 2026 旗舰芯片性能对比

## 项目说明

2026 年主流旗舰芯片（CPU/GPU/AI加速器）性能对比可视化页面，基于 Chart.js 渲染交互式图表。

## 文件结构

```
PROJECT_009_chip_comparison/
├── README.md                      ← 本文件
└── 2026_chip_comparison.html      ← 对比页面（可直接浏览器打开）
```

## 使用方式

直接用浏览器打开 `2026_chip_comparison.html` 即可查看。

## 技术栈

- HTML + CSS + JavaScript
- Chart.js 4.4.1（CDN 引入）
