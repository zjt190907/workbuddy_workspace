## 三、主人公设定（重写）

### 3.1 身份

**第一轮回的幸存者。** 兽人碎石部落的小头领，管二十来个人。第一轮文明末期，他没有保护好自己的子民——他的部落、他的兽人同伴、他认识的每一个人，全死在了巨熊之神疯狂的屠戮中。

他是唯一的活人。

不是因为他勇敢、特殊或被选中。也许只是神在彻底疯癫前最后那半秒的犹豫。也许只是巧合。但那道从掌心延伸到手腕的疤留了下来，而他也活了下来。

**永生不是天赋，是惩罚。** 他试过死——跳崖、走进火里、喝毒药——都不行。他的身体会自己修复。他活过了第一轮回的终结，因为某种连他自己都不理解的原因，他获得了"不死"——不是不老，而是很难死。很难很难。

### 3.2 核心创伤与人格

- **愧疚**："我没能保护他们。"这句话刻在他每一根骨头里。
- **孤立**：每一轮新文明都不认识他，不记得过去。他是活着的史书，但没人会翻。
- **自嘲**：活太久的人要么疯，要么学会笑自己。"一个拿着石斧去弑神的蠢货。"
- **疲惫的温柔**：他不是冷漠。他只是见过太多次。但他依然会在火堆边给新人类的孩子讲故事。

### 3.3 叙事声音

**口语化的第一人称。** 句式长短错落。有断裂。有停顿。有"放屁""你听清楚""算了"这种生活化的语气词。拒绝任何"在那遥远的过去"式华丽开腔。

用具体事物替代抽象抒情。不说"文明脆弱"，说"一阵风吹来就会熄灭"。不说"我很愧疚"，写"我翻了他的尸体，没翻到什么。一个兽人能有什么值钱的东西。"

### 3.4 情感弧线

```
第一轮结束 → 愧疚 + 愤怒 + 自杀意图
第二~五轮 → 麻木 + 自嘲 + 偶尔的共情燃起又熄灭
第六轮 → 被白厄触动 → 决定不再是旁观者 → 见证飞升
终局 → 白厄归乡 → 主人公释然
```

## 四、核心设定体系（术语修正版）

### 4.1 世界观规则

| 规则 | 说明 | 叙事功能 |
|------|------|----------|
| **黑潮** | 侵蚀星球，催生怪物 | 外部压力/倒计时 |
| **神明诞生** | 从生物思想中诞生（信仰最快，但不是唯一） | 核心矛盾 |
| **神明之爱** | 神无条件保护凡人，不分信与不信 | "爱"的具象化 |
| **神的疯狂** | 凡人发展科技/思想 → 神的不稳定性累积 → 满值后无差别屠杀 | 核心悖论 |
| **轮回** | 凡人死尽→神明消亡→黑潮新生→新轮回 | 结构骨架 |
| **成年礼** | 杀死本族神明→获得百年安宁，其间无新神诞生 | 文明成长仪式 |
| **飞升星空** | 对神的终极宣战。离开恒星引力即永绝神明。 | 终局 |
| **纪元碑** | 不受神明影响的古老造物，封存知识与文明记忆 | 跨纪元传承 |

## 七、叙事技法

### 7.1 去AI味执行原则

**禁止项（立即删除）：**
- "不仅仅……而且……" / "不仅……还……" 等工整对仗
- "不禁" / "不由得" / "仿佛"——删掉虚化抒情
- 万能金句式的段落结尾（"这就是……的意义。"）
- "某种意义上" / "在某种程度上"——要么说，要么不说
- 每段长度高度一致（三段整齐排比）→打乱节奏
- 过度解释人物情感（"他的内心充满了……"）→用行为替代

**鼓励项：**
- 句子长短不一，偶尔有断裂感
- 自嘲、黑色幽默的口语表达
- 感官细节（泥土的味道、血的温度、火的噼啪声）
- 对话简短但有信息量
- 允许不完美的、别扭的、磕磕绊绊的叙述
- 人物动作比人物心理描写更多

### 7.2 时间标记

- 每章关键节点标注年份（如"第三十七年"）
- 不会给精确的、统一的纪年体系——主人公的记忆本身就是模糊的

### 7.3 视角控制

- 全程第一人称
- 主人公可能提到"我后来才知道"——但不做上帝视角的预解释
- 白厄段落允许半第三人称观察（主人公目睹的方式）
