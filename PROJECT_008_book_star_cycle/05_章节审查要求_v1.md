# 《星潮》章节审查要求 v1

---

## 一、审查目标

为《星潮》各章节建立统一的审查标准，确保：
1. 世界观设定一致
2. 叙述视角正确（第三人称为主）
3. 文章基调准确（永生者的悲哀、文明与神的矛盾、勇气赞歌）
4. 人物刻画符合设定
5. 语言风格去 AI 味
6. 逻辑连贯、无文字错误

---

## 二、审查维度详解

### 2.1 世界观一致性（P0级）

**核心原则**：神因凡人思想而生，无主观意志，无条件爱世人

| 设定项 | 正确理解 | 错误理解 |
|--------|---------|---------|
| 神的本质 | 因凡人思想/信仰而诞生的存在，没有人的主观意志 | 神有思想、有意志、有喜怒哀乐、有人的判断力 |
| **神不沟通** | **神没有任何语言能力，不会说话、不会下命令、不会安慰、不会解释。神的"爱"只能通过动作表达（保护、猎食、战斗）** | 神开口说话（"你们受苦了""我会保护你们"等） |
| 神爱世人 | 无条件保护所有凡人，不分信徒与非信徒 | 有条件的爱，惩罚"冒犯者"、"不信者" |
| 神的杀戮 | 只在疯癫度满值后无差别屠杀，不分信与不信 | 主动下令杀人，有选择地杀"有罪者" |
| 疯癫机制 | 凡人发展科技/思想→疯癫度累积→满值失控 | 神因"贪婪"、"虚荣"、"自私"等人格化理由杀人 |
| 狂信徒处理 | 神清醒时为保护他们而让他们自裁 | 神主动命令狂信徒杀其他凡人 |

**审查检查清单**：
- [ ] **神是否在任何地方开口说话？**（说了→P0 违规）
- [ ] 神的行为是否有主观意志表达？（如判断"冒犯""贪婪""虚荣"→错误）
- [ ] 神的杀戮是否有选择性？（选择性杀戮→错误）
- [ ] 神的保护是否有条件？（有条件→错误）
- [ ] 疯癫机制是否正确呈现？（文明发展→累积→满值失控→正确）
- [ ] **神的"爱"是否通过动作而非语言表达？**（语言→违规，动作→正确）

**问题示例**：

❌ **错误示例**（第一章 v2）：
```
巨熊说：他冒犯了神。
巨熊说：他贪婪。
巨熊说：他虚荣。
巨熊说：他自私。
狂信徒说：请让我为你杀光所有不信者。
巨熊说：好。
```
**错误原因**：神有主观判断（贪婪、虚荣、自私），有选择性杀戮，命令狂信徒杀人

✅ **正确示例**（第一章 v1）：
```
神爱世人。
不分信徒和非信徒。它救人的时候，没问过你跪没跪过它。
那天晚上，神的疯癫度从六十二涨到了九十九。
因为望远镜。因为凡人看到了星空。因为凡人对星空产生了好奇。
当天夜里，神下令狂信徒集体自裁。
这是神在清醒时做的最后一件事。
它知道自己在失控。它在疯掉之前，先把最危险的人处理了。
```
**正确原因**：神无条件救人，疯癫度累积导致失控，清醒时保护狂信徒

### 2.2 叙述视角（P0级）

**核心原则**：第三人称为主（70-80%），第一人称为辅（20-30%）

| 叙述类型 | 占比 | 功能 | 适用场景 |
|---------|------|------|---------|
| 第三人称描述 | 70-80% | 展现事件、人物、场景、动作 | 场景描写、动作描写、其他人物 |
| 灰烬第一人称 | 20-30% | 推进故事、建立情感连接、揭示内心 | 灰烬的内心独白、回忆、感慨 |

**参考作品**：《穷鬼的上下两千年》
- 穿越者在历史中见证文明兴衰
- 大量第三人称客观描述，穿插穿越者的主观感受
- 时间跨度大，叙事节奏紧凑
- 历史厚重感与个人情感交织

**审查检查清单**：
- [ ] 第三人称段落占比是否达到 70-80%？
- [ ] 第一人称是否仅用于灰烬推进故事？
- [ ] 场景描写是否用第三人称？
- [ ] 宏大场面、史诗感段落是否用第三人称？
- [ ] 叙述视角切换是否自然流畅？

**叙述分工示例**：

✅ **第三人称**：
```
灰烬在河边砸鱼。
砸鱼这个词好听。其实就是往水里丢石头，丢中了算运气，丢不中就饿肚子。
碎石部落二十三个人，那天有十九个在河边上。
```

✅ **第一人称**（灰烬内心独白）：
```
我手上的疤是四千年前留下的。
说"四千年"其实不准确。我不太数年了。
第一轮文明灭亡之后，年这个数字对我来说就不太好用。
```

✅ **第一人称→第三人称切换**：
```
我是怎么活下来的？
想了好多年，没想明白。

灰烬活过了第一轮回的终结。
他不知道为什么。他试过死——跳崖、走进火里、喝毒药——都不行。
```

### 2.3 文章基调（P0级）

**核心基调**：永生者的悲哀 + 文明与神的矛盾 + 勇气赞歌

| 基调维度 | 具体表现 | 叙事功能 |
|---------|---------|---------|
| 永生者的悲哀 | 灰烬的愧疚、孤独、疲惫、自嘲 | 建立主人公情感弧线 |
| 文明与神的矛盾 | 神爱世人但因文明发展而疯癫 | 核心冲突，推动情节 |
| 先驱者之歌 | 每个时代都有先驱为文明之火而战 | 展现人类勇气与希望 |
| 勇气是人类的赞歌 | 凡人弑神、牺牲、选择、传承 | 主题升华 |
| 神爱世人 | 神无条件保护，疯癫后无差别屠杀 | 悲剧性、矛盾性 |

**审查检查清单**：
- [ ] 是否体现"永生者的悲哀"？（灰烬的愧疚、孤独、疲惫）
- [ ] 是否展现"文明与神的矛盾"？（神爱世人但因发展而疯癫）
- [ ] 是否有先驱者为文明之火而战？
- [ ] 是否体现"勇气是人类的赞歌"？
- [ ] 是否贯彻"神爱世人"的核心？

**基调体现示例**：

✅ **永生者的悲哀**：
```
我翻了他的尸体。没翻到什么。他身上没什么值钱的东西。一个兽人能有什么值钱的东西。
岩说"他饿"。
我答应给他烤鹿腿。
他没吃到。
```

✅ **文明与神的矛盾**：
```
神爱世人。
不分信徒和非信徒。它救人的时候，没问过你跪没跪过它。
但凡人每发展一项新技术——冶炼、锻造、造船、建城——神的疯癫度就往上涨一点。
```

✅ **勇气是人类的赞歌**：
```
我挥了一斧子。
石斧。砍在神的手臂上。
斧子碎了。
神的手臂上连痕都没留。
但我手上有了一道疤。
```

### 2.4 对话要求（P0级）

**核心原则：对话要"被说出来"，不要"被写出来"**

蛮荒时代的兽人、矮人、精灵、人类——他们的对话不应该干净、完整、文雅。

**必须包含的元素**：

| 元素 | 示例 | 适用场景 |
|------|------|---------|
| 语气词 | "干"、"操"、"妈的"、"放屁"、"扯淡" | 愤怒、恐惧、质疑 |
| 粗俗语言 | "狗日的"、"他娘的"、"吃屎去吧" | 兽人日常交流 |
| 不完整句 | "腿。腿没了。"（非"我的腿没了"） | 紧急、恐惧、受伤 |
| 打断/抢话 | 话说一半被另一个角色打断 | 紧张时刻 |
| 叫唤 | "走——！"、"别——！" | 生死关头 |
| 非语言声音 | "吼了一声"、"磨着牙"、"从喉咙里挤出一个声音" | 兽人特性 |
| 俚语/黑话 | 碎石部落内部特定说法 | 部落文化 |

**错误示例（公式化对话）**：

❌ 
```
黑子说："它们在干什么？"
灰烬说："拜神。"
黑子说："拜那个杀它们全族的东西？"
灰烬说："嗯。"
```

**正确示例（口语化对话）**：

✅ 
```
黑子说："它们搞什么？"
灰烬盯着那群跪在地上的兽人，看了很久。
"拜。"
"拜？"黑子的声音变了，"杀它全族的东西——拜？"
灰烬说："嗯。"
黑子没说话。过了一会儿骂了句脏话。灰烬没听清骂的什么。
```

**审查检查清单**：
- [ ] 是否有粗俗语言？（兽人对话中全无粗口→不合格）
- [ ] 是否有不完整句？（全部是完整句→不合格）
- [ ] 对话是否有停顿、打断、重复？（对答如流→不合格）
- [ ] 是否有非语言的声音描写？（全程只说话→不合格）
- [ ] **神是否开口说话？**（说了→P0 违规）

### 2.5 人物刻画（P0级）

**主人公：灰烬**

| 特质 | 表现 | 叙事功能 |
|------|------|---------|
| 愧疚 | "我没能保护他们"，刻在骨头里 | 核心创伤 |
| 自嘲 | "一个拿着石斧去弑神的蠢货" | 缓解沉重感 |
| 疲惫的温柔 | 见过太多次毁灭，但依然会给新人类的孩子讲故事 | 人性温度 |
| 孤立 | 每一轮新文明都不认识他，他是活着的史书，但没人会翻 | 永生者的孤独 |

**神明刻画**：
- 无主观意志，无喜怒哀乐
- 无条件保护所有凡人
- 疯癫度满值后无差别屠杀
- 清醒时可能保护狂信徒（让他们自裁）

**审查检查清单**：
- [ ] 灰烬刻画是否准确？（愧疚/自嘲/疲惫的温柔）
- [ ] 神的形象是否正确？（无主观意志/无条件爱世人）
- [ ] 先驱者形象是否有勇气和牺牲感？
- [ ] 配角行为是否符合人物设定表？

### 2.6 AI 味识别（P1级）

**禁止项**：
- "不仅仅……而且……"等工整对仗
- "不禁" / "不由得" / "仿佛"——虚化抒情
- 万能金句式段落结尾（"这就是……的意义。"）
- "某种意义上" / "在某种程度上"——模糊表达
- 每段长度高度一致的三段整齐排比
- 过度解释人物情感（"他的内心充满了……"）

**鼓励项**：
- 句子长短不一，偶尔有断裂感
- 自嘲、黑色幽默的口语表达
- 感官细节（泥土的味道、血的温度、火的噼啪声）
- 对话简短但有信息量
- 允许不完美的、别扭的、磕磕绊绊的叙述
- 人物动作比心理描写更多

**AI 味识别示例**：

❌ **AI 味示例**：
```
不仅仅是恐惧，更是一种深深的敬畏。
他的内心充满了愧疚和悲伤。
这就是文明轮回的意义。
```

✅ **去 AI 味示例**：
```
恐惧到了极点，不是恨。是跪。
我翻了他的尸体。没翻到什么。
一个兽人能有什么值钱的东西。
```

## 四、审查流程

1. **阅读全文**：通读章节，标注问题位置
2. **P0 检查**：世界观一致性、叙述视角、文章基调、人物刻画
3. **P1 检查**：AI 味识别、逻辑连贯性
4. **P2 检查**：风格统一性、文字错误
5. **生成报告**：填写审查模板，给出修改建议
