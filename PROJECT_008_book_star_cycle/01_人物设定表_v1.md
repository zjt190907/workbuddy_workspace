# 《星潮》人物设定表 v1

---

## 一、主人公

### 灰烬（Ash）

**身份**：第一轮回幸存者，兽人碎石部落小头领

**外貌**：
- 兽人特征：深褐色皮肤，厚实的肩膀，手掌宽大
- 左掌心到手腕有一道疤，像干涸的河
- 看起来三十岁左右，但实际活了六轮文明

**核心创伤**：
"我没能保护他们。"
第一轮文明末期，他的部落、他的兽人同伴、他认识的每一个人，全死在巨熊之神的屠戮中。他是唯一的活人。

**永生的性质**：
- 不是天赋，是惩罚
- 不是不老——而是很难死。跳崖、火烧、毒药，身体会自己修复
- 他试过自杀，试了很多次

**性格**：
- 愧疚：刻在骨头里的自责
- 自嘲：活太久的人要么疯，要么学会笑自己
- 疲惫的温柔：见过太多次毁灭，但依然会给新人类的孩子讲故事
- 偶尔的愤怒：对神的愤怒，对自己的愤怒

**叙事功能**：
- 引子部分第一人称，建立读者信任
- 主体故事第三人称，他是角色之一，不再是叙述者
- 最终章回归第一人称，完成情感收束

## 二、各轮回关键人物

### 第一轮回 · 巨熊时代

**巨熊（图腾：巨熊）**
- 第一位诞生的神
- 因兽人的恐惧和崇拜而诞生，无主观意志
- 爱世人——不分信与不信，无条件保护
- 疯癫机制：凡人发展科技/思想→疯癫度累积→满值后无差别屠杀
- 关键转折：望远镜让疯癫度从 62 暴涨至 99
- 清醒时最后一件事：让狂信徒集体自裁（保护他们）
- 结局：疯癫度满值后无差别屠杀，灰烬用石斧终结，文明随之全灭

**老祭司**
- 碎石部落的精神领袖
- 第一个发现"有东西在保护我们"的人
- 也是第一个被巨熊之神无意踩死的人

**碎石部落族人**
- 23人，最后只剩灰烬
- 名字不重要，重要的是他们的死亡顺序：
  1. 老祭司（被踩死）
  2. 7个去森林找浆果的（被熊吃）
  3. 河边的19人（神降临时被压死）
  4. 灰烬（活下来）

---

### 第二轮回 · 世界之树

**树神（图腾：世界树）**
- 从精灵的信仰中诞生
- 最温柔的神：为精灵提供食物、庇护、甚至氧气
- 悖论：太爱精灵→精灵开始崇拜→神的力量暴增→失控
- 结局：选择自裁，用自己的身体庇护最后的精灵

**瑟兰（Theran）**
- 最后的精灵
- 世界之树自裁后唯一的幸存者
- 绝望中看着自己的族群走向灭绝
- 灰烬试图救他，但他选择了留下

**精灵王**
- 下令所有精灵自杀的统治者
- 理由："神已经死了，我们没有活着的意义"
- 自杀前烧毁了所有精灵的历史记录

---

### 第三轮回 · 锻造时代

**铁王（矮人）**
- 第一个成功弑神的凡人
- 矮人锻造之城的建立者
- 发明了弑神武器："穿心枪"
- 性格：冷硬、务实、蔑视神
- 后期异化：从弑神的英雄，变成新神

**锻造之神（图腾：熔炉）**
- 从矮人的工业崇拜中诞生
- 能力：锻造一切金属，为矮人提供无尽矿藏
- 被工业化杀死——矿脉被掏空，神的力量源枯竭
- 死前最后一句话："你们已经不需要我了"

**铁王的儿子**
- 继承父亲的位置
- 试图阻止父亲的异化，失败
- 在父亲变成新神后，亲手杀死父亲

---

### 第四轮回 · 和平时代

**人王**
- 人类河谷王国的建立者
- 追求永久的和平与繁荣
- 性格：理想主义、温和、不愿流血
- 错误：与和平之神达成协议——没有战争，但也没有进步

**和平之神（图腾：白鸽）**
- 从人类对和平的渴望中诞生
- 能力：抑制一切暴力冲突
- 悖论：没有冲突→没有进步→文明慢性自杀
- 结局：被叛逆者联盟杀死，但文明已经来不及救了

**叛逆者首领**
- 人王的弟弟
- 看穿和平是慢性自杀的人
- 组织叛逆者联盟，杀死和平之神
- 但为时已晚——文明已经失去了所有反抗能力

---

### 第五回合 · 最短轮回

**狂信徒首领**
- 杀光所有不信者的人
- 建立了神权统治
- 相信"只有彻底的信才能避免神的愤怒"
- 结局：被自己制造的神杀死

**神之子（伪）**
- 狂信徒推出来的傀儡
- 没有任何真正的力量
- 被当作神的代言人，最后被神一巴掌拍死

**幸存的学者**
- 记录了最后时刻的人
- 留下一本《毁灭纪事》
- 灰烬后来找到了这本书

---

### 第六轮回 · 太阳时代

**白厄**
- 救赎者
- 百子中唯一成功弑神的孩子
- 性格：温柔、坚定、有超越年龄的成熟
- 在完成飞升后，选择归乡
- 名字来源："白圭之玷，尚可磨也；斯言之玷，不可为也"——《诗经》

**太阳之神（图腾：太阳）**
- 从所有种族的希望中诞生
- 最强大的神：为整个世界提供光明和能量
- 明知结局——知道凡人终将弑神——依然选择爱
- 最后的话："去吧。你们值得。"

**百子**
- 被神选中的100个孩子
- 每一个种族都有代表
- 白厄是其中之一
- 其他99个全部失败

**叛逆者联盟**
- 四族联合的弑神组织
- 灰烬在其中担任顾问（不是领袖）
- 提供了从过去轮回中积累的知识

## 三、跨轮回配角

### 3.1 岩（黑子）—— 第一轮回朋友

**身份**：碎石部落年轻猎手，灰烬最亲近的族人
**外貌**：黑色鬃毛，比其他兽人略矮，右手少了两根指头（被野兽咬掉的）
**性格**：嘴碎，总说丧气话，但在最危险的时候第一个冲上去

**与灰烬的关系**：
- 灰烬管他叫"黑子"，因为他头发黑得发亮
- 每次灰烬要做蠢事（比如跟着巨熊走三个月），他嘴里骂娘，脚还是跟着走
- 在望远镜事件后，他是第一批被疯癫的神踩死的人

**死亡场景**：
他倒在灰烬怀里的时候还在说话。他说"腿没了"。又说"有点冷"。又说"你说烤鹿腿什么时候能好"。

灰烬答应给他烤鹿腿。

他没吃到。

**叙事功能**：灰烬的核心创伤锚点。"我没能保护他们"——"他们"里第一个人就是黑子。每次灰烬手上的疤疼，想起来的是黑子的尸体的重量。

---

### 3.2 烙（Lao）—— 第三轮回结识的矮人学者

**身份**：矮人锻造之城的历史记录者
**外貌**：矮小但手指极长，左眼被铁水烫瞎，常年戴单片铜镜
**性格**：沉默寡言，但一旦开口就停不下来。唯一一个敢直接问灰烬"你到底是什么"的人

**与灰烬的关系**：
- 灰烬在第三轮回的矮人城市里流浪时遇见他
- 烙是第一个意识到灰烬不是"普通人"的凡人——他注意到灰烬不吃饭、不怕冷
- 灰烬向他坦白了自己的永生，烙用了一整夜的时间沉默
- 最后他说："那你的记忆就是这座城里最重要的矿脉。"

**结局**：
- 铁王封神后开始屠杀"知道太多的人"
- 烙没有逃。他把自己的记录凿在锻造之城的石墙上
- 他留给灰烬的最后一句话："帮我记住。"

**叙事功能**：
- 灰烬"活史书"身份的确认者——有了烙的托付，灰烬才真正意识到自己活着的意义不是惩罚，是见证
- 第三轮中最接近"理解灰烬"的人

---

### 3.3 影牙（Shadowfang）—— 跨轮回意识形态对手

**身份**：每一轮文明中都会出现的主张"彻底臣服于神"的人物，但每次都以不同面貌重生
**外貌**：每轮不同，但总有一道从左眉划到右颧骨的疤

**各轮回的形态**：
| 轮回 | 身份 | 主张 | 结局 |
|------|------|------|------|
| 第一轮 | 兽人狂信徒首领 | "唯有跪拜才能活" | 被神疯癫后屠杀 |
| 第二轮 | 精灵保守派祭司 | "神死之日即吾等死期" | 与精灵王一同自焚 |
| 第三轮 | 矮人反弑神派头目 | "神是造物主，不可杀" | 被铁王处决 |
| 第四轮 | 和平之神的首席祭司 | "没有神就没有和平" | 神死后精神崩溃 |
| 第五轮 | 狂信徒首领本人 | "杀光不信者" | 被自己制造的神杀死 |
| 第六轮 | 太阳神殿保守派 | "弑神是邪恶" | 在弑神之战中阵亡 |

**与灰烬的关系**：
- 灰烬在第二轮就发现了这个规律——每一轮都有一个人，面貌不同，但说的话一模一样
- 灰烬从不与他纠缠。活过六轮之后，他已经知道争论毫无意义
- 最后一轮，影牙临死前终于认出了灰烬——"你...还在？"
- 灰烬没有回答。这是他唯一一次后悔自己没有回答

**叙事功能**：
- 展现"臣服派"思想的顽固——每一轮文明都会重蹈覆辙
- 与灰烬的"变化派"形成对抗
- 最终轮回中的对视：六轮后的和解或永恒的对立

---

### 3.4 敏月（Minyue）—— 第六轮回的医生

**身份**：照顾百子的医师，人类，年纪约四十岁
**外貌**：灰白头发，手极其稳，但黑眼圈重得像被人打过
**性格**：沉默寡言到近乎阴郁，但对孩子极其温柔

**与灰烬的关系**：
- 灰烬带着第六轮的知识找到她时，她已经照顾了百子三年
- 她是唯一一个没有追问"你是谁"的人——她说"能帮忙就别废话"
- 99个孩子失败后，她瘦得只剩骨架，但还在照顾最后一个——白厄

**结局**：
- 白厄弑神成功后，她没有登船
- 她说："我的病人都在这里。"
- 灰烬后来发现她的墓——就在百子失败的那块空地上，她自己挖的

**叙事功能**：
- 展现"凡人英雄"——没有弑神之力，没有永生之命，但有愿意陪孩子走到最后的决心
- 灰烬承认："她比我勇敢。我第一次见死不救。她见了九十九次，还在救。"

---

### 3.5 单轮回配角（补充细节）

**第一轮回**：
- **老祭司**：碎石部落的精神领袖。白发几乎拖地，说话像个老糊涂，但每句话都是对的。第一个发现"有东西在保护我们"的人。死因：在巨熊脚下祈祷时被无意踩死。
- **碎牙**：碎石部落的老妇人，负责烤鹿腿。灰烬二十年还会梦到她烤的鹿腿——梦醒后什么都记不得，只记得那个味道和黑子没吃到。
- **七个去森林找浆果的**：名字已无人记得。灰烬只记得最后一个走出营地的是个十二岁的女孩，回头冲他笑了笑。

**第二轮回**：
- **瑟兰**（完整补充）：最后的精灵。世界之树自裁后唯一的幸存者。灰烬试图救他，他在灰烬背上挣扎了一下，然后不动了。他说："你不知道那是什么感觉。神是我们的父亲、母亲、太阳和雨。"灰烬确实不知道。他把瑟兰放下了。
- **精灵王**：瘦得像一把骨头，眼窝深得像两个洞。他下令精灵集体自杀时的语气平静得像在安排晚餐。灰烬问他为什么，他说："你见过失去太阳的植物吗？"

**第三轮回**：
- **铁王**（完整补充）：矮人，弑神者，后来的矮人之神。弑神前说的话："神算什么？连自己都保护不了。"封神后说的话："神算什么？连自己都做不了。"灰烬见过他最骄傲的样子，也见过他最疯的样子。他不知道哪一个才是真正的铁王。
- **铁王之子**：杀了自己父亲的人。弑父时没有哭。灰烬问他为什么，他说："我的父亲在他说'神算什么'那天就死了。后来的那个不是他。"

**第四轮回**：
- **人王**：人类河谷王国的建立者。一个笑起来像阳光的人。灰烬从未见过一个统治者如此真心地希望无人流血。但这正是问题——他希望无人流血，所以不敢做任何会让人流血的事。
- **叛逆者首领**（人王之弟）：亲手杀死和平之神的人。杀神前对人王说："哥，你太善良了。"杀神后看着文明慢性死亡说："我太迟了。"

**第五回合**：
- **幸存学者**：唯一在狂信徒屠杀中活下来的普通人。不勇敢，不聪明，只是运气好。他在绝望中写了一本书——《毁灭纪事》。灰烬后来找到这本书，发现里面全是用指甲刻的。"没有墨水了。"学者在结尾写道，"也没有人了。"
- **神之子（伪）**：一个十二岁的人类男孩，被狂信徒推上神位。他没有任何力量。他被神一巴掌拍死的时候，灰烬听见他喊了一声"娘"。

**第六轮回**：
- **白厄**（完整补充）：百子中唯一成功弑神的孩子。他是最后一个——前面九十九个都失败了。他的性格不像救世主，更像一个被责任压弯了脊梁的普通人。弑神成功后，他修复了生育中心，独自养育了无数孩子。飞升之后，他选择归乡。灰烬问他为什么，他说："星辰是文明的归宿。但我的弟弟妹妹还埋在下面。"
- **太阳之神**：从所有种族的希望中诞生。最强大的神，也是最清醒的神——他主动加入核聚变研究团队，让科学家研究自己的本体；他推广唯物主义，让信徒以科学而非信仰理解自己。灰烬不理解他。直到最后一刻，太阳之神对白厄说："去吧。你们值得。"灰烬才开始明白——这个神从一开始就知道结局。

```
    ↓ 见证
第一轮回 → 巨熊之神 → 老祭司 → 碎石部落（全灭）
第二轮回 → 树神 → 瑟兰（最后精灵）→ 精灵王
第三轮回 → 锻造之神 → 铁王（弑神者）→ 铁王之子
第四轮回 → 和平之神 → 人王 → 叛逆者首领（人王之弟）
第五回合 → 神之子（伪）→ 狂信徒首领 → 幸存学者
第六轮回 → 太阳之神 → 白厄（救赎者）→ 百子 → 叛逆者联盟
    ↓ 归乡
终局：白厄选择归乡，灰烬释然
```

## 五、命名规则

| 种族 | 命名风格 | 示例 |
|------|---------|------|
| 兽人 | 粗犷、有力、意象式单名或短名 | 灰烬、黑子（岩）、碎牙 |
| 精灵 | 柔和、诗意、带自然意象 | 瑟兰、艾拉 |
| 矮人 | 硬朗、金属感、带工艺意象，或简洁单名 | 铁王、烙、锻钢 |
| 人类 | 中性简洁，或带意象双字 | 人王、白厄、敏月 |
| 跨轮回者 | 名字跨轮回不变，或保留核心意象 | 灰烬（永存）、影牙（跨轮回转世） |

神明命名：直接用图腾命名，不加"神"字后缀
- 巨熊（不是巨熊之神）
- 世界树（不是树神）
- 太阳（不是太阳之神）
