# 《星潮》——文明轮回纪事

## 项目说明

科幻·文明史诗·轮回叙事长篇小说。

**核心命题**：神因爱而生，因爱而疯。文明每一次挣扎都在离星空更近一步。
**红线答案**：救赎不是逃离。是飞升之后，选择归乡。

| 项目 | 内容 |
|------|------|
| 书名（暂定） | 《星潮》 |
| 副标题 | 文明轮回纪事 |
| 类型 | 科幻·文明史诗·轮回叙事 |
| 视角 | 第一人称为主（70-80%），第三人称为辅（20-30%） |
| 预估体量 | 13万字（约12-15章） |
| 参考风格 | 《穷鬼的上下两千年》叙述手法 |
| 基调 | 永生者的悲哀 + 文明与神的矛盾 + 勇气是人类的赞歌 |

## 世界观核心规则（不可违背）

1. **神无主观意志**：因凡人思想而生，没有喜怒哀乐和主观判断
2. **神爱世人**：无条件保护所有凡人，不分信徒与非信徒
3. **疯癫机制**：凡人发展科技/思想 → 疯癫度累积 → 满值后无差别屠杀
4. **狂信徒处理**：神清醒时为保护他们而让他们自裁
