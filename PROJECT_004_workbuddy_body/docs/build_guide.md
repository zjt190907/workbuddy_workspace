### 打印步骤（自行打印）

1. 打开 `workbuddy_shell_v2.scad`，按需调整参数
2. 渲染（F6），导出 STL（F7）或直接导出
3. 在切片软件中导入 STL
4. 前壳添加树状支撑
5. 分别打印前壳和背板

## 组装步骤

### 第1步：烧录固件

1. 安装 Arduino IDE
2. 安装 ESP32 开发板支持（Board Manager 搜索 "esp32"）
3. 安装库：Adafruit GFX Library、Adafruit ST7735 and ST7789 Library
4. 打开 `firmware/workbuddy_body/workbuddy_body.ino`
5. 开发板选择 "ESP32C3 Dev Module"
6. 设置：USB CDC On Boot = Enabled, CPU Frequency = 160MHz
7. 连接 USB-C 数据线，上传固件
8. 验证：屏幕应显示 WorkBuddy 启动画面

### 第2步：接线

按照 `docs/wiring.md` 中的引脚映射表，用杜邦线连接屏幕和开发板。

**重要**：先接线，后通电！

### 第3步：试装屏幕

1. 将屏幕模块放入前壳的显示窗口
2. 确认屏幕居中、方向正确
3. 用 2 颗 M2×6mm 螺丝固定屏幕（通过边框孔位）

### 第4步：固定 ESP32

1. 在 ESP32-C3 SuperMini 背面贴一小块双面胶
2. 将 ESP32 粘贴在背板内侧
3. USB-C 接口朝向背板底部的线槽

### 第5步：理线

1. 将杜邦线沿壳体内部理顺
2. USB-C 数据线通过背板底部线槽引出
3. 确保线材不会挤压屏幕

### 第6步：合盖

1. 将背板对准前壳背面的卡扣位置
2. 轻压卡入，确认四角都卡紧
3. 如有松动，可用少量热熔胶辅助固定

### 第7步：测试

1. 连接 USB-C 供电
2. 观察启动画面和 WiFi 信息
3. 手机搜索 WiFi "WorkBuddy-Body"，密码 "buddy1234"
4. 浏览器访问 http://192.168.4.1
5. 测试各表情切换和画布功能

## 日常使用

1. 只需 USB-C 供电，无需电脑
2. 手机连 WiFi "WorkBuddy-Body" 后打开 http://192.168.4.1 控制
3. 注意：WiFi AP 模式下手机无法同时上网（可考虑后续改为 STA 模式接入家庭 WiFi）

## 进阶改装

- **WiFi STA 模式**：修改固件连接家庭路由器，手机可同时上网
- **Claude Code Hook**：参照 Claude Mochi 的 Hook 配置，将 WorkBuddy 的状态与 Claude Code 事件联动
- **传感器扩展**：ESP32-C3 还有 GPIO 0/5 可用，可接入温度/光照传感器
- **电池供电**：可加入 3.7V 锂电池 + TP4056 充电模块实现无线供电
